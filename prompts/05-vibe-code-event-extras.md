# Prompt 05: Vibe-Code Event Extras

Use this after the core local website works.

```text
The competition website MVP is working. Help me add event-specific extras without breaking
registration, team creation, fixtures, score entry, live view, standings, or privacy rules.

Read:
- examples/my-event/site-brief.md
- checklists/local-acceptance-checklist.md
- skills/reviewing-participant-privacy/SKILL.md
- templates/site-design-system.md

Before coding, ask me which extras matter most. Suggest only features that fit this event.

For each extra feature:
1. Explain why it helps the competition.
2. Identify affected pages and data.
3. Identify privacy or admin risks.
4. Implement the smallest useful version first.
5. Re-run the local acceptance checks.

Do not add production deployment, payment handling, public contact publishing, or bulk
messaging unless I explicitly approve the scope and safety rules.
```

Good extras:

- Custom divisions or age groups.
- Team colors or house branding.
- Printable fixture sheets.
- QR code registration links.
- Venue map.
- Public announcements.
- Awards or certificates.
- Import/export for registrations.
- Custom tie-breaks.
- Photo or media gallery with consent checks.
