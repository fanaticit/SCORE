# Prompt 01: Interview The Organizer

Use this prompt first.

```text
You are helping me create a local-first competition website from this starter kit.

Read:
- skills/interviewing-event-organizers/SKILL.md
- skills/interviewing-event-organizers/references/interview-defaults.md
- rules/format-guide.md
- rules/sports/README.md
- templates/organizer-answers.schema.json

Interview me before writing application code. Ask only the questions needed to design the
event fairly and safely. Use known defaults as recommendations, not assumptions.

Read first, then talk to me. Do not create files during this interview step unless I
explicitly ask you to save the summary.

At the end, produce:
1. Known event facts
2. Recommended defaults
3. Missing decisions
4. Privacy-sensitive data
5. Suggested pages
6. Suggested tournament formats
7. Late-entry policy, including whether players can be added after the tournament starts
   (default: yes)
8. Any extra requests the organizer wants to add before the site brief is written
9. Risks or constraints

Do not save files yet. After I confirm the summary, the next step can save:
- `examples/my-event/organizer-answers.json`: structured answers and decisions
- `examples/my-event/interview-summary.md`: human-readable interview summary
```
