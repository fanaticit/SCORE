# Publish As A New GitHub Repo

Use this folder as the root of the public project.

## Suggested Repo Name

`SCORE`

SCORE stands for Starter Competition & Organizational Resource Engine.

Other reasonable names:

- `competition-website-starter-kit`
- `tournament-vibe-coding-kit`
- `community-competition-builder`

## What To Include

Include everything inside `competition-starter-kit/`.

Do not include the old static sports-day website files unless they are added later as an
optional example implementation.

## First Public Commit

Recommended commit message:

```text
Initial competition website starter kit
```

## GitHub Description

```text
Starter Competition & Organizational Resource Engine for vibe-coding local-first competition websites.
```

## README Positioning

The project is not a website generator yet. It is a practical briefing kit for a coding
agent. The first output should be a local website tailored to the organizer's event.

## Suggested First Issues

- Add more sport rule references.
- Add generated examples for chess-only, football-only, and school house events.
- Add stack-specific implementation prompts for vanilla JS, React, Svelte, and Astro.
- Add tests for generated website acceptance criteria.
- Add a sample site generated from the kit as a separate `examples/generated-site/` folder.
